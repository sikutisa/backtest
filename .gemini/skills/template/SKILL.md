# 🎯 Standardized Expert Skill: [Skill Name]

## 📋 1. Purpose & Context
Briefly describe the domain expertise this skill provides and the specific problems it solves.

## 📥 2. Inputs & Prerequisites
Define the required context, files, or environment variables needed for this skill to function.

## ⚙️ 3. Standard Operating Procedure (SOP)
1. **Research:** Step 1 to understand the current state.
2. **Strategy:** Step 2 to formulate the plan.
3. **Execution:** Step 3 to apply the changes.

## ✅ 4. Validation & Quality Control
Specific tests or commands to verify the correctness of the output generated by this skill.

## ⚠️ 5. Exception Handling
Known edge cases, potential errors, and how the agent should react to them.

## 🔄 6. Evolution Protocol
How this skill should be updated or deprecated as the project evolves.
